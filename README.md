# TwilioSmsSender

**TwilioSmsSender** is a web application that sends an sms to a verified Twilio number.

## Libraries/Api used

- [Twilio](https://www.twilio.com/en-us)


## How to test

- Clone the project.
- Open the project in Visual Studio.
- Add your Twilio account info in the config.py file
- Run the file app.py
